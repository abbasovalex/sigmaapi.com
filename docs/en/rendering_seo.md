# Rendering for SEO

Sigma API helps pre-render your Vue, React, Angular, or other JavaScript-based apps for faster indexing on Google, Yandex, and beyond.

#### It’s very easy to use — let’s get started!
1. Get your API key by [signing up](https://sigmaapi.com/accounts/register/) now.
2. Then choose how to render your pages: [via API](/rendering_via_api) or with the built-in [UI scheduler](/rendering_via_ui).
